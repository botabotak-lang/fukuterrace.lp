# fukuterrace-LP

This folder was created for the new landing page project.

You can place your HTML/CSS/JS files here.

## キャッシュ対策（重要）

`index.html` や `thanks.html` では、CSS/JS を読み込む際に `?v=20251026-2` のようなクエリでバージョンを付与しています。スタイルやスクリプトを更新した際は、**必ず HTML 内のバージョン番号も更新**してください。ここを変更し忘れると、ブラウザが旧ファイルをキャッシュから読み込み、新しい変更が反映されません。日付＋枝番など分かりやすい値に置き換えて運用してください。
