# LPコンバージョン率改善のためのMECE要件定義書

現在、LPのファーストビュー（FV）改修が完了しました。
次のフェーズとして、FVを通過したユーザーを確実に「見学予約・資料請求」へコンバージョン（CV）させるための改修を行います。

**CV改善の方程式:**
$$CV = (動機 - 不安) \times 行動のしやすさ$$

この方程式に基づき、以下の4つのカテゴリー（MECE）で実装・コンテンツ追加を行ってください。

## 1. 【動機付け（Incentive）】「今」動く理由の強化

FVで興味を持ったユーザーに対し、先延ばしを防ぐためのオファーと緊急性を追加します。

- [ ] **オファーの具体化（CTA周りのマイクロコピー）**
    - 単なる「資料請求」ではなく、メリットを提示する。
    - 文言例：「失敗しないお墓選びガイドブック（PDF/冊子）プレゼント」
    - 文言例：「ご自宅でゆっくり検討できる『家族会議用パンフレット』を郵送」
- [ ] **限定性と緊急性の明示**
    - 「損をしたくない」心理に訴求する。
    - 文言例：「第1期区画の日当たり良好エリアは残りわずかです」
    - 文言例：「オープン記念価格は**◯月◯日まで**」

## 2. 【信頼・証拠（Trust & Proof）】「騙されない」保証の提示

高額商材特有の疑り深さを払拭するコンテンツを追加します。

- [ ] **スタッフの顔が見える化（ザイオンス効果）**
    - 住職だけでなく、「現地案内スタッフ」の笑顔の写真を追加する。
    - 「強引な営業は一切しません」という宣言文をスタッフ写真と共に掲載する。
- [ ] **比較検討への先回り（優位性の証明）**
    - ユーザーが脳内で行う比較を可視化する「比較表」セクションを実装する。
    - **比較対象:** 福てらす墓園 vs 市営墓地 vs 一般的な納骨堂
    - **比較項目:** 総額（隠れコストなし）、バリアフリー、承継者の要不要、管理の手間
    - ※自社が圧倒的に有利に見える項目を選定すること。

## 3. 【不安払拭（Risk Reduction）】ネガティブ要素の排除

申し込み直前の心理的ブレーキを外します。

- [ ] **「隠れコスト」の完全否定**
    - 料金セクションに強い見出しを追加する。
    - 文言例：「後から追加費用は1円もかかりません」（※管理費の扱いは注釈）
    - 「寄付金の強制なし」「檀家義務なし」を強調表示する。
- [ ] **アクセスの「心理的距離」短縮**
    - 地図だけでなく、車での来園しやすさをアピールする写真ブロックを追加する。
    - コンテンツ：「駐車場からお墓までの歩数（近さ）」、「運転が苦手な方でも安心の広い進入路」。

## 4. 【操作性（Usability / EFO）】行動ハードルの最小化（最重要）

フォーム入力の面倒さをシステム的に解決し、EFO（入力フォーム最適化）を行います。

- [ ] **フォームへの導線・機能の分離**
    - 1つのフォームで混在させず、目的別に入り口（またはフォームの挙動）を分ける。
    - **資料請求（ハードル低）:** 入力必須項目を「名前・住所・電話番号」のみにする。「希望日時」は聞かない（非表示）。
    - **見学予約（ハードル高）:** 「希望日時」を選択させる。
- [ ] **フローティングボタン（追従CTA）の最適化**
    - スマホ表示時、画面下部に固定されるボタンを見直す。
    - まだ検討段階のユーザーのために、「電話」よりも「資料請求（無料）」ボタンを最も押しやすい位置・大きさに設定する。
